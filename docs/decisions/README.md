# 결정 기록

이 폴더는 프로젝트의 중요한 기술 결정과 제품 결정을 기록하는 곳이다.

## 기록 대상

- 데이터 구조 변경
- Firebase 컬렉션 추가 또는 필드 의미 변경
- Vite/React 전환 같은 큰 구조 변경
- 보고서 계산 기준 변경
- 학부모에게 표시되는 평가 기준 변경

## 작성 형식

파일명은 `YYYY-MM-DD-short-title.md` 형식을 사용한다.

```markdown
# 결정 제목

## 배경

왜 결정이 필요했는지 적는다.

## 결정

무엇을 선택했는지 적는다.

## 영향

기존 기능, 데이터, 사용자 경험에 어떤 영향이 있는지 적는다.

## 대안

검토했지만 선택하지 않은 대안을 적는다.
```
